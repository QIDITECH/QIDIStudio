# QIDIStudio
QIDIStudio is a professional 3D printer slicing softwareï¼Œwhich is perfectly compatible with all printers and 3D printing filaments of QIDI Technology. Multi-platform support, simple inerface, easy to use, complate functions, easy to learn 3D printing.
